# 📖 [LOSANGELES-MOUNTAINS] <a name="LOSANGELES-MOUNTAINS"></a>

**[LOSANGELES-MOUNTAINS]** This repository contains code for a skills test at Coalition Technologies. The page is hyking experience where users can see the mountains, the teams and the schedule for hyking.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Setup

Clone this repository to your desired folder using the following command; git clone https://github.com/Amen-Musingarimi/coalition-technologies-skills-test.git

### Run tests

To run tests, run the following command: npx hint . and npx stylelint "\*_/_.{css,scss}" and npx eslint .

You can deploy this project using Github pages.
